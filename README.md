## CRUD TASKS BACKEND 
Para ejecutar el proyecto lo que se debe hacer es:
**1-**Ingresar al link del repositorio, clonar el proyecto en gitbash usando el comando "git clone" + url del repositorio
**2-**Una vez que ya se haya clonado, para abrir la carpeta en gitbash usar el comando cd + "nombre del archivo".En caso de querer ver el codigo de el proyecto hacer click izquierdo en el archivo y seleccionar "abrir con Visual code" o de otra manera abrir Visual Code, ir a file y seleccionar la opcion "Open folder" donde pueden seleccionar la carpeta del proyecto.
## Para poder ejecutar el proyecto haciendo usos de sus funcionalidades debe hacer lo siguiente:
-iniciar el archivo en visual code, abrir la terminal e iniciar node con el comando "npm init -y"
-luego descargar las dependencias necesarias: "npm i express" y "npm i msql2"
-Como el proyecto se trata de la conexion de un base de datos a un servidor con funcionalidades CRUD,debe importar la base de datos en la plataforma PHPmyadmin para que el servidor pueda reconocerlo y pueda ser capaz de acceder y modificar la base datos
